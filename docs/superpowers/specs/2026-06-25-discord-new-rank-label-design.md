# Discord 신규 기록 순위 표시 설계

## 목표

전체 도전장 로그 채널에 등수 상승 메시지를 보낼 때, 기존 DX 스코어가 없던
차트의 순위 변동을 `신규 -> #현재순위`로 표시한다.

## 판별 규칙

- `previousDxScore === null`: 신규 기록
- 그 외: 기존 기록 갱신

신규 기록이라도 `previousRank`는 랭킹 이벤트와 월별 상승 정도 집계에 사용할 수
있으므로 변경하지 않는다.

## 변경 범위

- 전체 도전장 로그의 `buildChannelRankUpMessages` 출력만 변경한다.
- 신규 기록은 `순위: 신규 -> #N`으로 표시한다.
- 기존 기록은 현재와 동일하게 `순위: #N -> #M`으로 표시한다.
- 개인 채널의 등수 하락 메시지, DB 스키마, 랭킹 이벤트 계산은 변경하지 않는다.

## 검증

- 신규 기록 이벤트가 `신규 -> #현재순위`로 출력되는 단위 테스트를 추가한다.
- 기존 기록 이벤트가 기존 숫자 형식을 유지하는 테스트를 보존한다.
- 전체 테스트와 프로덕션 빌드를 실행한다.
